# CT FastAPI Boilerplate

## Stack
- FastAPI
- Pydantic
- Uvicorn
- uv (dependency management)
- Ruff (linting)
- pre-commit (code quality hooks)
- Makefile (command abstraction)

## Setup

python -m uv sync

## Run

python -m uv run uvicorn app.main:app --reload
